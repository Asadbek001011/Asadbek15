Cascading Style Sheets CSS

CSS veb-sahifa yaratyvchilari tomonodon rang, shiriftlari, alohida bloklarning tartibini va ushbu veb-sahifalarning paydo bo`lishining boshqa jihatlarini belgilash uchun ishlatidi. CSS-ni ishlad chiqishninq asosiy maqsidi veb-sahfalaning (HTML yoki boshga belgilash tillari yordamida ishlab chiqarilgan) mantigiy tuzilishinig tavsifini ushbu veb-sahifa (hozir rasimlar CSS tili yordamida ishlab chiqarilgan) tashqi ko'rinishi tovsifidan ajratish edi. Ushbu ajratish hujjatning mavjudlingin oshirish, ko'proq moslashuvchanlargi va toqdimotni boshqarish qobiliyatini ta'minlad, shuningdek tarking tarbdagi murakkabilk va tokrorlanuchanlikni mumkin
